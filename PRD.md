# EVENTIA - Prompt

## Plataforma EVENTIA - App de Eventos y Experiencias

Aplicación web tipo app móvil para descubrir, crear y gestionar eventos, con sistema de roles, suscripciones, tickets y chat en tiempo real.

---

## Objetivo

Implementar una app navegable con datos mock que permita:

- Explorar eventos
- Comprar tickets
- Crear eventos (organizadores)
- Interactuar mediante chat
- Gestionar suscripciones

---

## Roles

- Administrador
- Organizador (por suscripción)
- Usuario normal
- Usuario premium

---

## Funcionalidades

- Registro con correo y contraseña
- Preferencias iniciales
- Edición básica de perfil
- Visualización y búsqueda de eventos
- Creación y edición de eventos
- Compra y visualización de tickets
- Chat en tiempo real (eventos y privado)
- Notificaciones
- Geolocalización
- Favoritos
- Personalización UI (premium)

---

## Pantallas

- Inicio
- Explorar
- Detalle del evento
- Chat
- Crear evento
- Editar evento
- Perfil
- Mis eventos
- Mis tickets
- Notificaciones
- Suscripciones
- Configuración
- Admin

---

## Notas técnicas

- Datos mock en memoria
- Persistencia en localStorage
- UI tipo app móvil
- SPA sin recargas
- Preparado para Supabase
